## Features of Monkey Lang
- C-like syntax
- Variable Bindings
- Integers and Booleans
- Arithmetic expressions
- Built-in functions
- First-class and Higher-order functions
- Closures
- String data structure
- Array data structure
- Hash data structure

## Features of the Interpreter
- Lexical Analysis
- Parsing
- Evaluation
- Macro System
